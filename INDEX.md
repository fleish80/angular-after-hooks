## 🔍 Quick Links

### APIs Covered
- **afterNextRender** - One-time DOM initialization
- **afterEveryRender** - Continuous render synchronization (renamed from `afterRender`)
- **afterRenderEffect** - Reactive DOM updates with signals
- **effect** - General signal effects

### Source Code
All demo components are in:
```
apps/main-app/src/app/demos/
├── presentation.ts              # Main presentation
├── after-next-render-demo.ts    # afterNextRender examples
├── after-every-render-demo.ts   # afterEveryRender examples
├── after-render-effect-demo.ts  # afterRenderEffect examples
└── effect-demo.ts               # effect examples
```
